Object Hierarchy

  GObject
   +----ArvFakeCamera

Description

ArvFakeCamera is a class that simulate a real camera, which provides methods for the implementation of ArvFakeDevice and ArvFakeStream.

arv-fake-gv-camera is a GV camera simulator based on this class.
